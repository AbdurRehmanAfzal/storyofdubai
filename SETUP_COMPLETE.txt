CRITICAL INFORMATION
================================================================================

Business Rules (NEVER violate):
  1. Never delete scraped data — only mark is_active=False
  2. Never scrape without 2+ second rate limiting
  3. Never hardcode API keys — always use environment variables
  4. Never generate duplicate content — check similarity before saving
  5. Never break URL structure — changing slugs destroys SEO
  6. Always include schema.org JSON-LD on every page
  7. Always update PROGRESS.md when completing major tasks
  8. Scoring output must be deterministic (same input = same score)
